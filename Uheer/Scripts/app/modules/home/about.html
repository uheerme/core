About uheer.me

uheer.me is a awesome place where you can listen to music uploaded by other people or share your own playlist! It's ideal for creating silent-disco or throwning low-costing parties, as it plays synchronously in all devices and every one can listen to it with their headsets or use their phones as music boxes!

uheer.me was created in 2015 and it had its code made available in 2015. Please check out our organization on GitHub if you want to contribute somehow: github.com/uheerme.

How to use it?

Sign in and go to channels. You can create your own or search for an existing one.

You've selected an existing channel:

  • Just seat back and relax. The song will start playing automatically.

You've created your very own channel:

  • Use the dashboard to upload songs and manage your channel.
  • Share your channel with your friends and have fun!

Please notice that once a channel is deactivated, all songs are deleted and it cannot be restored again.

What is an API and why do you guys have it?

API stands for Application Program Interface and it's the reason why a single web service can synchronize different platforms, like web browsers and Android phones. uheer.me has an open API, which means if you are not satisfied with our apps, you can create your own and use our service! To understand more how things work, take a look at the /API's docs.

How does the synchronization work?

  • A DJ creates a channel, uploads songs and start one of them.
  • Multiple peers visit that DJ's channel.
  • The server keeps import information about the channel, such as which music is playing and when it has started. Peers' computers collect this information and synchronize themselves with the server by executing Christian's algorithm multiple times.
  • Periodically, the peers' computers will estimate how synchronized they are, in comparison to the server. If required, they will attempt to improve synchronization by calculating the errors gotten so far and estimating translations for the current song, adapting in each cycle.
